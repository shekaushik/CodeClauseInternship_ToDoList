# Project Title:

Todo List Web Application 

## 1. Project Description: 

Todo List is a web application that performs the following functionalities:

- User login
- Add a todo
- List all todo's
- Delete a todo
- User logout

## 2. Tech Stack: 

- Java 8
- Java JSP / Servlet
- Maven
